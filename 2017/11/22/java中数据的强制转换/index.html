java中数据的强制转换

发表于 2017-11-22 | 分类于 代码积累集 ， java

自动转换

程序执行时自动执行，基本类型由低级到高级分别为(byte，short，char)--int--long--float--double
一般时低位数类型向高位数类型转换，转换关系如下：
![转换关系](../../public/images/TIM图片20171212210058.png)

— 对于byte,short,char三种类型而言，他们是平级的，不能相互自动转换，需要使用强制转换

强制转换

(转换后的类型)转换前的类型
**可能会发生精度丢失**

包装类过渡类型转换

所谓包装类，就是可以直接将简单类型的变量表示为一个类
Java共有六个包装类，分别是Boolean、Character、Integer、Long、Float和Double；分别对应于 boolean、char、int、long、float和double。而String和Date本身就是类

简单类型的变量转换为相应的包装类，可以利用包装类的构造函数。即：
Boolean(boolean value)、Character(char value)、Integer(int value)、Long(long value)、Float(float value)、Double(double value)

在各个包装类中，总有形为<数据类型>Value()的方法
例如，intValue()，调用该方法可以得到整型变量。

字符串型与其它数据类型的转换

调用toString()方法，基本所有java.lang.Object类派生的所有类均提供该方法

常用实例

String转int

int num1 = Integer.parseInt(number);//使用Integer的parseInt方法
int num2 = new Integer(number);//强制转换
num3 = Integer.valueOf(number).intValue();//先转Integer类型，再调用intValue()转为int

String转float和long与上述类似

String转byte[]

使用String类自带的getBytes()方法

byte[]转String

byte[] number = "121121".getBytes();
String num1 = new String(number);

path类型转String

String fileName=path.getFileName().toString();

int转String

String num1 = Integer.toString(number);//使用Integer的toString方法
String num2 = String.valueOf(number);//使用String的valueOf方法
String num3 = "" + number;//强制转换

long-int互转

int num2 = new Long(number).intValue();// 调用intValue方法

long num3 = Long.valueOf(number);

其实他俩用普通的强制转换就可以了。
